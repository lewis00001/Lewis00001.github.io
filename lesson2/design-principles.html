White Space

COLLY.COM

The white space of a design is important because it provides a clean and professional appearance. It also provides a place for the eye to rest and enhances the visibility of important content. 

The colly.com site uses white space very well.

The page features many sections of text, as well as images. However, the content does not seem cluttered or overwhelming. The flow and logical order of the content is very clear. The use of white space allows the site to feel comfortable.

Contrast

KAHKOW.DO

An effective use of contrast can aid in the usability of a website. Contrasting colors, shapes, and sizes can allow a user to get oriented with the content and quickly locate  the most important information.

The kahkow.do site uses contrast in an effective way.

The page above uses contrast to direct the eye of the user. The detail of the product image draws the eye first because the rest of the site is clean and simple. The next place the eye is drawn is to the active filter button so the user can find the product they're looking for. The third contrasting item is the chat bubble - incase they need assistance ordering.

Repetition

SIMPLIFILE.COM

In design, repetition can be a powerful visual way to unify elements. Elements of a webpage can be unified through text, color, shape, etc. It allows for a strong cohesive presentation for information.

Simplifile.com has used many elements of repetition.

The site lists several products, each on its own card. Every card uses a colored icon from the same palette with the same art style. Each has a title in the same font followed by a dividing line and description. Also, all the cards have a 'Learn More' button in the same color as the icon.
