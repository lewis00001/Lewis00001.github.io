Site Name

The site name chosen for this project is Weatherlicious. It's a cool combonation of the word 'weather' and 'delicious'. It extends from the idea that good weather allows a person to go out and have a yummy pinic, and bad weather lets them stay home and have a yummy picnic.

Site Purpose

The purpose of this site is not only to provide current weather information, but to serve as a go-to place for event promotion and local news. It will cover a region including Preston, Soda Springs, and Fish Haven, Idaho. There is a chance that a few people living in these places might get access to the internet someday and we want to be ready for them!

Domain Name

The domain URL will be 'weatherlicious.com' to match the name of the website.
